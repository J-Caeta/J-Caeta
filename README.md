### 🚀 Projeto em Destaque: Lunnar Pet Health (LPH)

Um ecossistema completo de monitoramento e prontuário inteligente para saúde de pets, desenvolvido com foco em alta manutenibilidade, design premium e práticas robustas de segurança.

**O que estou construindo neste projeto:**
* **Arquitetura:** Aplicação Fullstack desacoplada (React 19 + NestJS 11) sob padrões corporativos.
* **Backend:** NestJS 11 (Node.js) com TypeORM e banco de dados MySQL.
* **Frontend:** React 19 com Vite, TypeScript e interface baseada em *Glassmorphic design*.
* **Segurança:** Validação estrita via DTOs, prevenção contra XSS armazenado e mitigação de IDOR (UUIDs).
* **Features Avançadas:** Dashboard modular, calculadora de ração, diário fisiológico e gerenciador de consultas.
